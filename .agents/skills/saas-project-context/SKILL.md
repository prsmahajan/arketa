## Project: Studio & Wellness SaaS Platform

### What it is
A SaaS platform combining two products under one codebase:
- **Vertical 1 — Studio Management**: For pilates studios, yoga studios, fitness studios. Class scheduling, booking, member management, payment collection.
- **Vertical 2 — Wellness Professionals**: For nutritionists, dietitians, fitness coaches. Client management, session booking, payment collection in-app.

Inspired by: Aghor.com (studio side) + WellnessZ.in (wellness pro side). Not a clone — a better, more premium version targeting a gap between these two.

### Revenue goal
$5,000 USD/month recurring. This is the north star. Every decision (pricing, niche, feature priority) is filtered through this.

### Who built this / builder profile
- Paras, 23, Pune, India
- BCA graduate, sales background, learning performance marketing (Google Ads, Meta Ads)
- AI-assisted development: Claude Opus + Gemini + GPT-4 as the dev stack
- Solo founder, no technical co-founder

### Current build stage
- Design phase (Figma/UI mockups)
- Backend + DB work starts after design is complete
- No live product yet

### Target customers

**Studio vertical ICP:**
- Boutique pilates/yoga/fitness studio owners
- Affluent US metro cities (NYC, LA, SF, Miami, Chicago)
- 1–3 locations, 50–500 active members
- Pain: Mindbody is too expensive/complex, simpler tools lack payment features
- Budget: $99–$299/month SaaS

**Wellness pro vertical ICP:**
- Independent nutritionists, dietitians, online fitness coaches
- Primarily US-based, some India-based (secondary)
- Pain: Using 3–4 tools (Calendly + Stripe + Google Sheets + WhatsApp) patched together
- Budget: $29–$99/month SaaS

### Design aesthetic direction
Premium, modern, boutique wellness feel. Reference: the dashboard mockup uses sage green as accent, DM Serif Display for headings, flat minimal cards. NOT generic purple-gradient SaaS. Think: something a boutique studio owner in a US metro would feel proud to use.

### Tech decisions (AI-assisted build)
- Frontend: React (component-based, AI-generated)
- Backend: TBD (likely Node.js or Supabase)
- Payments: Stripe integration planned
- DB: TBD

### Key features to build (priority order)
1. Studio/coach profile setup + onboarding flow
2. Class/session scheduling
3. Client/member management
4. Payment collection (Stripe)
5. Analytics dashboard (sales, visits, memberships)
6. Marketing tools (email, referrals)

### What NOT to build yet
- Mobile app
- Multi-location enterprise features
- Complex reporting
- Marketplace/discovery features
